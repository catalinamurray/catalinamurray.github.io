NATURAL LANGUAGE PROCESSING

For The Aerospace Corporation


Topic Modeling


Utilized various NLP tools in a service arcitecture with API endpoints. Used Docker to streamline the services.


Due to contractor proprietary information I can't share any data or specificties about the project. However the basic struture utilized BERTopic


The data is in the form of sentences and is preprocessed to remove any unecessary characters. Next the numerical embeddings are created for the data using transformers that keep the semantic meanings of the data. The dimension is then reduced and final topics are created using a clustering algorithm. To better understand the topics KeyBERT is used to extract the most respresentative and meaningful words from each topic.

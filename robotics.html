ROBOTICS PROJECTS


TURTLEBOT 4


Utilized NAV2 and SLAM for autonomous mapping with the Turtlebot4 robot. I also created a tutorial as an introduction into this method, which includes some other ROS2 functions to do wall following and other commands.


Tutorial

ROMI


Built the Romi robot with the Romi 32U4 Control Board, LEDs, accelerometers, gyroscopes, buzzer, and LED screen. Utilized the ROMI robot and PlatformIO to create various wall following, line following and obstacle avoidance commands. Eventually created a script for the robot to follow a specific path using these commands.

Line Following Intersection Crossing

UNDERWATER


Utilized the Blue Robotics ROV2 for various applications. Deployed it in the lab tank to better understand how the robot functions. Utilized the BlueRobotics Ping Echosounder and Altimeter as a sonar to map the tank.
